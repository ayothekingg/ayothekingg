# React 19: What's new? 
Being the most popular JavaScript framework, [React](https://code.pieces.app/blog/react-18-a-comprehensive-guide-to-the-latest-features-and-updates) continually rolls out improvements and new exciting features to further ease the process of web development. Some of the features that accompanied the React 18 version include automatic batching, new server-side rendering APIs, new strict mode, and more. The React 19 version, announced at the React Conf 2024, has more exciting features, one of which is the open source [React Compiler](https://react.dev/learn/react-compiler). 

In this blog post, we'll dive into the exciting features of the newly released React 19.
